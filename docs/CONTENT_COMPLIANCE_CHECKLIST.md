# Content Compliance Checklist

Use this checklist before publishing or updating every product page.

## Product Page Checklist

- [ ] No medical claims.
- [ ] No disease, symptom, cure, or condition promises.
- [ ] No intoxicating, recreational, smoke-shop, or party language.
- [ ] Non-intoxicating positioning is included.
- [ ] CBG/hemp wording is accurate for the product.
- [ ] COA link is included or marked TODO.
- [ ] Batch number is included or marked TODO.
- [ ] Ingredients are included or marked TODO.
- [ ] Cannabinoid amounts are included only when label/COA supported.
- [ ] COA/lab status is accurate.
- [ ] Adult-use language is included.
- [ ] Disease disclaimer is included.
- [ ] Shipping/compliance note is included.
- [ ] No child-oriented candy presentation or copy.
- [ ] No unsupported customer review claims.
- [ ] Product title, SEO title, and description use safe wellness language.

## Required Product-Page Language

- Non-intoxicating.
- CBG-rich wording is supported by the product label, batch details, or COA.
- COA Available, COA Pending, Batch Pending, or Not Applicable status is accurate.
- Batch-labeled or batch number pending.
- Adult use only.
- Not intended to diagnose, treat, cure, or prevent disease.

## Required Launch Review

- [ ] Real product photo added.
- [ ] Real label text added.
- [ ] Real ingredients added.
- [ ] Real batch number added.
- [ ] Real COA linked.
- [ ] Real shipping restrictions confirmed.
- [ ] Legal/compliance review completed.
- [ ] Payment-method/platform review completed.
